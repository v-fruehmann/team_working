Проект_6: Предсказание стоимости автомобилей по их характеристикамПроект выполнялся командой Slava (https://www.kaggle.com/c/sf-dst-car-price-prediction/slava_team), в которую вошли: Сергей Фадеев(fadeevsu@gmail.com), Варвара Фрюманн(v.fruehmann@ gmail.com), Слава (apc0808@yandex.ru)Цель проекта: обучить лучшую модель для предсказания цен на автомобили на auto.ruПроект разбит на несколько частей: - Парсинг данных с сайта auto.ru и формирование тренировочного датасета на основе тестовых данных- Обработка и анализ данных, в т.ч. обогащение данных- Эксперименты ML, работа с метрикой MAPE- Submission на KaggleОтдельно прикладываем ноутбук с кодом для парсинга и ноутбук с решением.
