Цель проекта: обучить лучшую модель для предсказания цен на автомобили на auto.ru

- Парсинг данных с сайта auto.ru и формирование тренировочного датасета на основе тестовых данных- Обработка и анализ данных, в т.ч. обогащение данных- Эксперименты ML, работа с метрикой MAPE- Submission на KaggleПрикладываем решение:

1. ноутбук с кодом для парсинга 'parcing.py'
2. ноутбук с решением 'car_price_prediction.ipynb'
